This is simple weather widget app that uses [OpenWeatherMap](https://openweathermap.org/) API to fetch weather data.
It's part of a Next.js tutorial available on [wheresbaldo.dev](https://wheresbaldo/tech/nextjs/build-a-weather-widget-using-next-js).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
